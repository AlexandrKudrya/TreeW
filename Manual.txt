Инстрункия по установке:
1) Установите python 3.8+
https://www.python.org/downloads/

2)Установить библеотеки из файла requirements.
pip istall -a requirements.txt

3)Запустите файл main.py

Назначение файлов:
1) main.py - главный файл
2)папка data хранит модели пользователей, вопросов и игр. А также файл db_session.py, отвечающий за создание сессий работы с балой данных
3)папка db хранит базу данных
4)папка game_logs - содержит логи игр
5)папка static отвечает за хранение статических файлов
6)папка templates отвечает за хранение html разметки
